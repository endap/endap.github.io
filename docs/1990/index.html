equations

$\ \ \ \ \ \ \ \ $

$$\tag{1} X(\omega) = \int_{-\infty}^{+\infty} x(t) e^{-j \omega t} \ dt $$

$\ \ \ \ \ \ \ \ $

$$\tag{2} X(\omega) \approx \sum_{n=0}^{\infty} x(t_n) e^{-j \omega t_n} \ \Delta t $$

$\ \ \ \ \ \ \ \ $

$$\tag{0} e^{j \omega t} = \cos \omega t + j \sin \omega t $$

$\ \ \ \ \ \ \ \ $

$$\tag{4} x(t) = 3 \sin 20\pi t + 4 \cos 5\pi t + 1 $$

matrix
