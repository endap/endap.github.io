Steps to install Python on macOS
Just a new experience

It is my first time to install Python on macOS. Previously, I am able to install it on Windows 11, both on local computer and also remote one using AnyDesk .

Open Terminal

Find the Terminal icon on the Dock.

image-01

Click and hold until the context menu appears as follows.

image-02

Choose New Window and click it.

image-03

Now you can start to type in the Terminal

Type in the Terminal as follows.
