1-d uniform-linear-motion

  • Sparisoma Viridi
30 May 2026 · 2 mins read · edit

Uniform linear motion is the first topic in mechanics 1. Linear motion is motion in one spatial dimension 2, while uniform motion occurs when object velocity remains constant throughout the motion 3. In a uniform linear motion object moves along a straight line with constant velocity 4. Using Python and Matplotlib graph of $x-t$ can be drawn for various values of velocity and initial position 5. One-dimesion, or 1-d, informs that the straight line is along $x$ direction. In general, motion in 1-d can have acceleration or non-constant velocity 6.

equations

$$\tag{1} a = 0. $$

$$\tag{2} v(t) = v_0. $$

$$\tag{3} x(t) = x_0 + v_0 (t - t_0). $$

notes

  • It is common to choose $t_0 = 0$ for simplification.
  • Initial velocity is velocity at initial time $v(t_0) = v_0$.
  • Initial position is position at initial time $x(t_0) = x_0$.
  • There is an explicit notation that uses $x$ index, e.g. $r_x \equiv x$, $v_x \equiv v$.
  • There is also another notation that uses 1 for $x$, e.g. $x_1 \equiv x$, $v_1 \equiv v$.
  • The use of such indices is convenient for advancing the system to 2-d or 3-d.
