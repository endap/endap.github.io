equations

  • It supports equations via KaTeX.
  • This is an inline one $y = ax^2 + bx + c$.
  • This is an diplayed one $$\tag{1} x_{1,2} = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}. $$
  • The $x_1$ and $x_2$ in Eqn (1) are roots of $y = f(x)$.
